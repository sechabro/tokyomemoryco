This is an example **Photo Blog** post — delete it once you've seen how it works.
Because it's an `.mdx` file, you can mix Markdown with the same photo components
the galleries use. Put images in `src/assets/blog/` and reference them by
filename — no `import` needed.

A single full-width photo:

<Photo src="meadow-path.jpg" caption="A path through the meadow" />

Photos go through the exact same pipeline as the galleries: responsive WebP,
capped on the long edge, and click-to-open in the full-screen lightbox.

A two-up gallery — captions default to the filename, just like the main galleries:

<Gallery photos={["still-water.jpg", "forest-edge.jpg"]} />

Write as much or as little prose around them as you like. When you're ready,
swap these placeholders for your own photos and start your first real post.
